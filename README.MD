# DMS

- DMS: Datasource Management Service
- DM: Datasource Module


## Dependency

- Kotlin 1.3.5
- Springboot 2.1.9
- Gradle 5.6.2


## Description

This project is a datasource management service of **Cloud Database** based on sdk development

Following datasource would be developed as an example:
- mysql (5.7, 8.0)
- sqlserver
- redis
- mongodb
